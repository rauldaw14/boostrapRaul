Propiedades CSS para Imágenes

Background Image

La propiedad background-image permite establecer una imagen de fondo en un elemento.

  • background-position: Define la posición.
  • background-size: Controla el ajuste de la imagen.
  • background-attachment: Determina si se desplaza con la página.

Object Fit & Position

Podemos modificar cómo se visualiza una imagen usando object-fit , object-position , object-fit: cover;.

Bordes y Sombras

Podemos agregar bordes y sombras a imágenes para mejorar su presentación visual.

border y box-shadow

Filtros en Imágenes

Podemos aplicar filtros a las imágenes usando CSS, como escala de grises, desenfoque o brillo.

filter: grayscale(100%);

Animaciones en Imágenes

Podemos animar imágenes con transiciones y efectos en CSS.

transition: transform 0.5s ease-in-out;

Otras Propiedades CSS para Imágenes

Existen otras propiedades CSS que pueden transformar la apariencia de las imágenes, proporcionando efectos y estilización adicional. Algunas de estas propiedades son:

  • clip-path: Permite recortar imágenes en formas específicas, como círculos o polígonos.
  • opacity: Controla la opacidad de una imagen, permitiendo que sea más transparente.

Ejemplos:

clip-path

clip-path: circle(50%);

opacity

opacity: 0.2;
